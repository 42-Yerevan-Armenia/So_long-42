# So_long-42

### Commands :

| Key | Action |
|---|---|
| <kbd>W</kbd>, <kbd>↑</kbd>| Up |
| <kbd>S</kbd>, <kbd>↓</kbd>| DOwn |
| <kbd>A</kbd>, <kbd>←</kbd>| Left |
| <kbd>D</kbd>, <kbd>→</kbd>| Right |
| <kbd>ESC</kbd>, `⎋`| Exit |
---

### Structure of the map :

| Symbol | Description|
|--|--|
| `0` | Floor |
| `1` | Wall |
| `P` | Player |
| `E` | Exit |
| `C` | Coin |
| `M` | Enemy |
